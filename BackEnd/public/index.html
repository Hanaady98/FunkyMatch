API Overview

Welcome to the API for managing users and cards. Below is a brief description of the available endpoints:

üîë Authentication & User Routes

  • POST /users/register: Register a new user (Public)
  • POST /users/login: Login an existing user (Public)
  • GET /users/: List all users (Admin)
  • GET /users/:id: Get full user profile (Owner/Admin)
  • GET /users/public/:id: Get public profile data (Public)
  • PUT /users/:id: Update profile details (Owner/Admin)
  • PATCH /users/:id: Change user role (User/Moderator/Admin) (Admin)
  • DELETE /users/:id: Delete account (Owner/Admin)

For more information, visit the Users Postman documentation.

üí¨ Chat & Messaging

  • GET /users/private-messages/:roomId: Get chat history (Participants)
  • GET /users/:id/private-messages: List all conversations (Owner)
  • POST /users/:id/send-private-message: Send a private message (Owner)

For more information, visit the Chat & Messaging documentation.

üõ°Ô∏è Admin Moderation

  • POST /users/admin/ban: Ban user (temporary/permanent)
  • DELETE /users/admin/ban/:userId: Remove user ban
  • GET /users/admin/bans/:userId: View user's ban history

For more information, visit the Admin Moderation documentation.

üéØ Hobby System

  • GET /users/hobbies/all: List all available hobbies
  • PATCH /users/:id/add-hobby: Add hobby to profile
  • PATCH /users/:id/remove-hobby: Remove hobby from profile
  • GET /users/members/:hobby: Find users by hobby

For more information, visit the Hobby System documentation.

üìù Posts

  • POST /posts: Create a new post (Registered users only)
  • GET /posts: Get all posts (Admin only)
  • GET /posts/user/:id: Get posts by user (Profile Owner)
  • GET /posts/public/user/:id: Get posts by user (Public)
  • PUT /posts/:id: Update a post (Post Owner)
  • DELETE /posts/:id: Delete a post (Post Owne)
  • PATCH /posts/:id: Like or unlike a post (Registered Users)

For more information, visit the Posts documentation.
